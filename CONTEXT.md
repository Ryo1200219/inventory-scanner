# 在庫管理Webアプリ：実装詳細指示書 (Inventory Scanner Pro)

## 1. プロジェクト目的
スマートフォンでJANコード（バーコード）をスキャン、または商品名であいまい検索し、Googleスプレッドシートの在庫数とマスターデータをリアルタイムに更新・管理する。

## 2. システム構成
* **Frontend**: HTML5, Tailwind CSS, JavaScript (Quagga2ライブラリ)
* **Backend**: Google Apps Script (GAS) - Web Appとして公開
* **Database**: Google Spreadsheet (2枚のシート構成)

## 3. スプレッドシート仕様
### シート1: 「在庫集計」
* **A列**: JANコード (キー)
* **B列**: 商品名
* **C列**: 在庫数

### シート2: 「在庫リスト」
* **A列**: タイムスタンプ
* **B列**: JANコード
* **C列**: 商品名
* **D列**: 変動数量
* **E列**: 操作後の在庫数

## 4. バックエンド実装要件 (code.gs)
* **doGet(e)**:
  * 在庫集計データをJSONで返す。
* **doPost(e)**:
  * LockServiceによる排他制御。
  * 在庫集計の更新と、在庫リストへの履歴追記。

## 5. フロントエンド実装要件 (index.html)
### スキャン精度向上
* 解像度 1280x720 推奨。
* 3回連続一致バッファロジック。
* 横長ガイド枠と減光オーバーレイ。

### 検索・フィルタリング
* あいまい検索窓によるリアルタイムフィルタリング。
* リストタップからの手動更新。

### UI/UX
* ハーフモーダルによる更新画面。
* 成功時のトースト通知。

## 6. ライブラリ
* Quagga2
* Tailwind CSS

## 7. エージェントへの特記事項
* GASのURLは定数として定義。
* タップターゲットを大きく確保。
* 適切なエラーハンドリング。
