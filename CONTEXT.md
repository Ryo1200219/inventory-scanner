# 在庫管理スキャナー：カメラレイアウト最適化の実装指示

## 1. 運用ルール（重要）
- **バージョン管理**: 本指示を適用後、現在のアプリバージョンを確認し、末尾を必ずインクリメント（+1）して更新すること。


## 2. 目的
スマホカメラの物理的な位置（端末上部）と、画面上のプレビュー位置を近づけることで視覚的な視差を解消し、近距離スキャン時のピント合わせを容易にする。

## 3. 実装詳細 (index.html)

### A. プレビュー領域のスタイル
`#interactive.viewport` を横長（height: 180px）に設定し、上部の角丸を排除して画面上端に密着させる。
```css
#interactive.viewport {
    position: relative;
    width: 100%;
    height: 180px;
    overflow: hidden;
    border-radius: 0 0 1.5rem 1.5rem;
    background: #000;
}
```

### B. レイアウト構造
HTMLの最上部にカメラプレビューを配置し、その下に操作パネルを配置する。
1. 最上部: `#interactive` (カメラ)
2. その下: ヘッダー、検索、リスト

### C. ガイド枠
JANコードの形状に合わせ、横長でスリムなスキャンガイドを表示する。

## 4. 追加要件
- スキャン解像度は 1280x720 を維持すること。
- キーボードの自動閉鎖（Enterでのblur）を維持すること。
