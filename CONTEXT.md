# 実装依頼：検索ロジックの高度化（表記揺れの吸収）

現在のプログラムに対して、検索の利便性を向上させるための「曖昧検索ロジック」を実装してください。

## 1. 実装要件

検索バー（#search-input）での検索時に、以下の表記揺れを区別せず、同一の文字としてヒットするようにロジックを修正してください。

* ひらがな と カタカナ: （例：「めがね」と「メガネ」を同一視）
* 全角英数字 と 半角英数字: （例：「１２３」と「123」を同一視）
* 大文字 と 小文字: （例：「Apple」と「apple」を同一視）
